This is a VC10 Wizard to setup a default project to create a _2Real Plugin.

Some things you have to do:

* Change the ABSOLUTE_PATH parameter in _2Real Plugin.vsz.
* move 
	_2Real Plugin.vsz
	_2Real Plugin.vsdir and
	_2Real Plugin.ico
	to C:\Users\USERAME\Documents\Visual Studio 2010\Wizards.

* Then open Visual Studio 2010 and you will be able to select _2Real Plugin from the "Wizard" menu.
